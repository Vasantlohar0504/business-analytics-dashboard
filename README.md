# 📊 Business Analytics Dashboard

## 📌 Project Overview

This project analyzes retail sales data to uncover meaningful business insights. Using Python for data analysis, SQL for querying business metrics, and Power BI for visualization, the project helps identify sales trends, customer behavior, and product profitability.

The goal is to transform raw sales data into actionable insights for better business decision-making.

---

## 🛠 Tools & Technologies
* **Python**
* **Pandas**
* **Matplotlib**
* **Seaborn**
* **SQL**
* **Power BI**

---

## 📂 Dataset
This project uses the **Sample Superstore Dataset**, which contains retail sales data including orders, customers, products, regions, sales, and profit information.

---

## 🔍 Analysis Performed

### Sales Trend Analysis
* Analyzed monthly and yearly sales trends
* Identified top-performing regions and products

### Customer Segmentation
* Analyzed sales by customer segments
* Identified high-value customers

### Profit Analysis
* Evaluated profit by product category and sub-category
* Identified loss-making products

### Regional Sales Performance
* Compared sales performance across regions
* Identified the most profitable markets

---

## 📊 Visualizations

The project includes various visualizations such as:

## 📈 Power BI Dashboard

An **interactive Power BI dashboard** was created to visualize key business metrics including:

This project demonstrates how data analysis and visualization can help businesses:
